## GNOME Web (epiphany)

This is the GNOME Web browser, formerly known as 'epiphany'.

To build and run, you need the [webkitgtk41](../webkitgtk/webkitgtk41/) package
from this repo.  This package includes support for both JIT and libsoup3.
